# First User Save Guide

이 문서는 개발자가 아닌 실제 사용자가 처음 `EmailPilotAI.exe`를 열었을 때 필요한 개념만 따로 설명한다.

## 1. 세이브 파일이란?

- 세이브 파일은 파일 하나가 아니라 폴더 전체다.
- 이 폴더 안에 아래 데이터가 같이 쌓인다.
  - 앱 설정
  - 메일 동기화 결과
  - 리뷰 상태
  - 운영 workbook
  - 로그
- 기술적으로는 `workspace.epa-workspace.json` 파일이 들어 있는 폴더지만, 사용자에게는 “세이브 파일 폴더”로 설명한다.

## 2. 새 세이브 파일을 만들면 어디가 달라지나?

- 새 세이브 파일 만들기를 실행하면, 선택한 폴더 안에 앱이 필요한 구조를 자동으로 만든다.
- 사용자는 내부 구조를 외울 필요가 없다.
- 중요한 점은 하나다.
  - 나중에 이 폴더를 통째로 복사하면 설정과 결과도 같이 이동한다.
- 새 세이브 안에는 `mail`, `exports`, `logs`, `secure`, `state`, `locks` 폴더가 자동으로 준비된다.
- 예전 테스트에서 보이던 `참고자료`, `실행결과`, `기대되는 산출물` 폴더는 새 세이브에서 더 이상 만들지 않는다.

## 3. 실행 파일과 세이브 파일은 다르다

- 실행 파일:
  - 항상 Windows 로컬 경로의 `D:\EmailPilotAI\portable\EmailPilotAI\EmailPilotAI.exe`
- 세이브 파일:
  - 로컬 폴더도 가능
  - 공유 폴더도 가능

즉, 앱은 로컬에서 실행하고, 세이브 파일은 필요하면 공유 폴더에 둘 수 있다.

## 4. 왜 처음에는 빠른 테스트 동기화를 먼저 하나?

- 실제 메일함에는 처음부터 수천 건이 들어 있는 경우가 많다.
- 그래서 처음부터 전체를 돌리면 오래 걸리고, 설정이 틀렸을 때 확인도 늦어진다.
- 기본 권장 순서는 아래다.
  1. 세이브 파일 만들기
  2. 설정 입력
  3. 계정 연결 확인
  4. 빠른 테스트 동기화
  5. 결과가 괜찮으면 전체 동기화

빠른 테스트 동기화는 최근 10건만 가져와서 연결, 분류, workbook 반영이 자연스럽게 이어지는지 먼저 확인한다.

## 5. 로그인 ID는 무엇인가?

- 대부분의 계정은 이메일 주소 그대로 로그인하면 된다.
- 하지만 일부 메일 서비스는 이메일 주소와 다른 로그인 이름을 요구할 수 있다.
- 그래서 앱에서는 기본적으로 이메일 주소로 로그인하고, 필요할 때만 고급 설정의 `로그인 ID`를 따로 넣는다.

## 6. INBOX는 무엇인가?

- `INBOX`는 대부분의 메일 서비스에서 기본 수신함을 가리키는 이름이다.
- 계정 연결 확인이 성공하면 앱이 실제 폴더 목록을 읽고, 가능한 경우 `INBOX` 또는 그에 해당하는 폴더를 기본값으로 추천한다.
- 사용자는 처음부터 직접 폴더 이름을 외워 입력할 필요가 없다.

## 7. 엑셀 양식은 무엇인가?

- 엑셀 양식은 결과를 어떤 Excel 구조에 맞춰 쓸지 알려주는 기준 workbook이다.
- 새 세이브 파일을 만들면 기본 엑셀 양식이 자동 준비되므로, 대부분의 사용자는 이 값을 직접 바꿀 필요가 없다.
- 특별한 양식이 있을 때만 고급 설정에서 다른 엑셀 양식으로 바꾼다.

## 8. 세이브 파일 폴더를 복사하면 무엇이 같이 이동하나?

- 같이 이동하는 것:
  - 저장된 설정
  - 이미 받아온 메일 bundle
  - 리뷰 상태
  - workbook 결과
  - 로그
- 같이 이동하지 않는 것:
  - Windows 실행 파일 자체

즉, 세이브 파일 폴더는 “작업 상태”, exe는 “앱 프로그램”이다.

## 9. 이 PC에서 자동으로 기억하는 것

- 이 PC에서는 아래 정보가 로컬 암호화 저장소에 따로 저장될 수 있다.
  - 마지막으로 연 세이브 파일 폴더
  - 마지막 세이브 파일 암호
  - 기본 OpenAI API key
- 그래서 앱을 다시 열면 마지막 세이브가 자동으로 다시 열리거나, 새 세이브를 만들 때 API key가 자동으로 채워질 수 있다.
- 이 정보는 세이브 파일 폴더를 복사해도 다른 PC로 같이 이동하지 않는다.

## 10. 권장 사용 흐름

1. `D:\EmailPilotAI\portable\EmailPilotAI\EmailPilotAI.exe` 실행
2. 새 세이브 파일 만들기 또는 기존 세이브 파일 열기
3. 설정에서 이메일 주소, 비밀번호, API key 입력
4. 계정 연결 확인
5. 빠른 테스트 동기화
6. 리뷰센터 확인
7. 문제 없으면 전체 동기화
