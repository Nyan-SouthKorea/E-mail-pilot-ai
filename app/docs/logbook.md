## 현재 스냅샷

- 로컬 FastAPI 서버와 pywebview launcher가 있다.
- 워크스페이스 열기/생성, 설정, 통합 리뷰센터, 동기화 시작 버튼이 있다.
- 리뷰센터는 sqlite state DB 기준으로 triage, dedupe, 대표 export 상태를 보여준다.
- 원본 열기 링크는 workspace 상대경로를 기준으로 OS 파일 열기를 시도한다.
