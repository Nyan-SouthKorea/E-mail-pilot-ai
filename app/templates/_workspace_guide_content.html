세이브 파일이란?

세이브 파일은 이 앱의 데이터가 쌓이는 폴더 전체입니다. 이 폴더 안에 설정, 받은 메일, 리뷰 상태, 엑셀 결과가 함께 저장됩니다.

  • 기존 세이브 열기: 이미 만든 세이브 파일 폴더를 다시 엽니다.
  • 새 세이브 만들기: 저장 위치와 이름만 정하면 앱이 새 폴더를 자동으로 만듭니다.
  • 세이브 파일 폴더를 통째로 복사하면 설정과 결과도 같이 이동합니다.

어떤 폴더를 고르면 좋을까?

  • 새 세이브는 기본 제안 위치를 그대로 쓰거나, 원하는 상위 폴더를 고르면 됩니다.
  • 기존 세이브는 `workspace.epa-workspace.json` 파일이 있는 폴더를 고르면 됩니다.
  • `repo` 폴더나 이전 베타 구조 폴더는 새 세이브 파일로 사용할 수 없습니다.

실행 파일과 세이브 파일의 차이

  • 앱 실행은 Windows 로컬 `D:\EmailPilotAI\portable\EmailPilotAI\EmailPilotAI.exe`에서만 합니다.
  • 세이브 파일은 로컬 폴더나 공유 폴더 어디든 둘 수 있습니다.
  • 공유 폴더에 세이브를 두면 다른 환경에서도 같은 상태를 이어서 볼 수 있습니다.

처음에는 어떻게 시작하나?

  1. 새 세이브 파일을 만듭니다.
  2. 설정에서 이메일 계정과 API key를 입력하고 계정 연결 확인을 실행합니다.
  3. 빠른 테스트 동기화로 최근 10건만 먼저 확인합니다.
  4. 결과가 맞으면 전체 동기화로 넘어갑니다.
